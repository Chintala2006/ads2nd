# Global Temperature Analysis and Climate Trends

## Project Overview
This project analyzes global temperature data to understand long-term climate trends and patterns. The analysis focuses on identifying temperature changes over time, seasonal variations, and evidence of global warming.

## Dataset
The project uses publicly available climate datasets:

- GlobalTemperatures.csv
- GlobalLandTemperaturesByCity.csv

These datasets include:
- Date (dt)
- Average Land Temperature
- Temperature Uncertainty
- City and Country-level temperature data

## Objectives
- Analyze global temperature trends over years
- Identify seasonal temperature variations
- Perform time series analysis
- Check stationarity of temperature data
- Visualize climate patterns

## Tools and Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- Statsmodels

## Project Workflow
1. Data Loading and Cleaning
2. Handling Missing Values
3. Exploratory Data Analysis (EDA)
4. Feature Engineering (Year, Month, Season)
5. Seasonal Analysis
6. Time Series Analysis (ADF Test)
7. Visualization of Trends

## Key Insights
- Global temperatures have shown an increasing trend over the years, indicating global warming.
- Seasonal temperature patterns show consistent variation across years.
- Time series analysis helps evaluate whether temperature data is stationary.

## Note
This project was developed as part of a university assignment and demonstrates data analysis and time series analysis techniques using real-world climate data.
